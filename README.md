Requirements:
 - Android 7.0 and Up
 - Substratum/OMS Support ROM
 - Substratum Theme Manager (Get on Google Playstore)

 How To Install:
 - Download theme
 - Open Substratum Theme Manager
 - Select theme Overlays
 - Apply
 - Restart SystemUI
 - Done
